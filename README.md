# mcdc javascript library

## Install

```
npm install mcdc-js
```

## Import

CommonJS

```
var mcdcJS = require('mcdc-js');
```

Front end

```
<script type="text/javascript" src="/browserify-mcdc.js"></script>

# 或者
<script type="text/javascript" src="/browserify-mcdc-min.js"></script>

# 访问
console.log(window.mcdcJS)
```
